# Notable Lists
## Proyecto Final de programación Aplicada 2
Crea notas desde cualquier lugar, desde cualquier dispositivo y ¡compartelas con tus amigos para editar juntos!
**Funcionalidad**:
- CRUD de notas
- Crear usuarios
- Añadir amigos
- Compartir notas
- Editar notas live

**Tecnologías/filosofías utilizadas**
- MVI
- Clean Architecture
- Offline First
- Material 3 expressive
- Jetpack Compose
- Kotlin

Demonstración: https://youtu.be/hJqVZdMoNco
